### Requirements
1. Python 3.7
2. Pymoo 0.5.0
3. PyQt5 5.9.2
4. [MATLAB Online](https://matlab.mathworks.com/)

### Running

Navigate to the HA8/code folder. Type the following in the command window:
```bash
chmod +x main.sh
./main.sh
```

Weighted Sum Initialization uses the fmincon in MATLAB. Open MATLAB.
